3DеИ§жНЃ

еѓєдЇО3DзЪДжЛУжЙСиґЕеѓЉдљУпЉМеПѓдї•еИ©зФ®дЄАдЄ™жХіжХ∞зЪДжЛУжЙСдЄНеПШйЗПжЭ•жППињ∞

$$N_w=\frac{1}{24\pi^2}\int d^3k\epsilon^{ijk}\text{Tr}\left[Q^\dagger_{\bf k}\partial_i Q_{\bf k}Q^\dagger_{\bf k}\partial_j Q_{\bf k}Q^\dagger_{\bf k}\partial_k Q_{\bf k}\right]$$

$$\delta_{nk}\equiv\langle n,{\bf k}\rvert\mathcal{T}\Delta^\dagger_{\bf k}\rvert n,{\bf k}]\rangle$$

ињЩдЄАй°єдї£и°®зЪДжШѓеЬ®жЯРдЄ™иГљеЄ¶дЄКпЉМзФµе≠РйЕНеѓєзЪДе§Іе∞ПгАВжЦЗзЂ†дЄ≠иЃ®иЃЇзЪДйГљжШѓеЉ±йЕНеѓєжЮБйЩРпЉМдєЯе∞±жШѓиѓізФµе≠РйЕНеѓєдїЕдїЕжШѓеЬ®иієз±≥йЭҐеЊИе∞ПзЪДиМГеЫіеЖЕињЫи°МпЉМжЙАдї•иЗ™зДґе∞±жШѓењљзХ•дЇЖиієз±≥йЭҐдєЛйЧізЪДзФµе≠РйЕНеѓєпЉМиАМйАЪеЄЄжГЕеЖµдЄЛпЉМиієз±≥йЭҐйГљжШѓзФ±дЄАжЭ°зЛђзЂЛзЪДиГљеЄ¶иі°зМЃзЪДпЉМжЙАдї•ињЩйЗМдєЯе∞±дїЕдїЕиАГиЩСдЇЖеЄ¶еЖЕзЪДзФµе≠РйЕНеѓєгАВ

ж†єжНЃжЦЗзЂ†дЄ≠зЪДиЃ®иЃЇпЉМжЬАеРОеПѓдї•е∞ЖињЩдЄ™$N_w$еМЦзЃАдЄЇ

$$N_w=\frac{1}{2}\sum_s\text{sgn}(\delta_s)C_{1s}$$

$$\begin{equation}\begin{aligned}C_{1s}&=\frac{1}{2\pi}\int_{\text{FS}s}d\Omega^{ij}\left[\partial_ia{sj}({\bf k})-\partial_ja_{si}({\bf k})\right]\
a_{si}&-i\langle s{\bf k}\rvert\partial/\partial k_i\rvert s{\bf k}\rangle\end{aligned}\end{equation}$$

ињЩйЗМзЪД$s$дї£и°®зЪДжШѓиієз±≥йЭҐзЪДзіҐеЉХпЉМдєЯе∞±иѓійЬАи¶БеѓєжЙАжЬЙиієз±≥йЭҐзЪДиі°зМЃж±ВеТМгАВ$\text{sng}(\delta_s$)дї£и°®зЪДжШѓ$\delta_{nk}$еЬ®зђђ$s$дЄ™иієз±≥йЭҐдЄКзЪДзђ¶еПЈгАВе¶ВдЄКйЭҐзЪДиЃ®иЃЇпЉМиієз±≥йЭҐж≠§жЧґзЫЄдЇТзЛђзЂЛпЉМжѓПдЄ™иієз±≥йЭҐдЄКйГљдЉЪе≠ШеЬ®зФµе≠РйЕНеѓєгАВиАМеЕЈдљУзФµе≠РйЕНеѓєзЪДзђ¶еПЈе∞±еПЦеЖ≥дЇОдљУз≥їзЪДиієз±≥йЭҐдљНзљЃеТМзФµе≠РйЕНеѓєељҐеЉПгАВ$C_{1s}$жШѓжЬАзЖЯжВЙзЪДChernжХ∞пЉМеРМж†ЈзЪДжИСдїђйЬАи¶БеѓєжѓПдЄ™зЛђзЂЛзЪДиієз±≥йЭҐиЃ°зЃЧеЕґChernжХ∞гАВеЬ®3DдљУз≥їдЄ≠пЉМиієз±≥йЭҐдЄКзЪДзФµе≠РйЕНеѓєзђ¶еПЈеТМиієз±≥йЭҐдЄКзЪДChernжХ∞еЕ±еРМеЖ≥еЃЪдЇЖжПРз§ЇжШѓеР¶дЄЇжЛУжЙСгАВињЩжШѓдЄАдЄ™йЭЮеЄЄзЃАеНХзЪДеИ§жНЃпЉМеП™йЬАи¶Бж†єжНЃж≠£еЄЄжАБзЪДиГљеЄ¶иЃ°зЃЧеТМиґЕеѓЉжАБзЪДзФµе≠РйЕНеѓєеИЖжЮРе∞±еПѓдї•еЊЧеИ∞дљУз≥їжШѓеР¶дЄЇжЛУжЙСзЪДгАВ

еѓєдЇО2DдљУз≥їпЉМеЕґжЛУжЙСдЄНеПШйЗПжШѓ$Z_2$пЉМж≠§жЧґеИ§жНЃеПШеЊЧжЫіеК†зЃАеНХ

$$N_{2D}=\Pi_s\left[\text{sgn}(\delta_s)\right]^{m_s}$$

a 2D TRI superconductor is nontrivial (trivial) if there are an odd (even) number of Fermi surfaces each of which encloses one TRI point in the Brillouin zone and has negative pairing.
